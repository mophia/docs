##  1、springboot 的简单概述

### 1. 概念

- 随着动态语言的流行，java显得配置繁琐，开发效率低、项目部署复杂、继承第三方技术难度大
- **spring boot** ： `约定大大于配置`

### 2、优点

- 快速搭建项目
- 主流开发框架都提供了污配置集成，内置了配置
- 项目可以独立运行，无需单独配置servlet容器，内置了容器 tomcat ，可以更换
- 提供运行时监控，如日志等
- 与云原生有天然集成

3、缺点

- 由于配置都是内置，报错时定位比较困难
- 版本迭代速度快，版本改动比较大
